# Projetos de validação para Pingu

Este diretório é um playground de regressão manual para linguagens mapeadas no runtime do Pingu.

## Estrutura

- `javascript/`
- `python/`
- `elixir/`
- `go/`
- `rust/`
- `ruby/`
- `lua/`
- `vim/`
- `c/`
- `terraform/`
- `yaml/`
- `markdown/`
- `mermaid/`
- `dockerfile/`
- `shell/`
- `toml/`

Cada linguagem possui pelo menos um projeto em `project-*` para validar comentários acionáveis, geração de snippets e criação de contratos.

## Como usar os projetos



- abra um projeto com a extensão da linguagem correspondente;
- use comentários acionáveis do Pingu (`//:`, `#:`, `#::`, etc);
- valide no terminal com os comandos do README local de cada projeto.
